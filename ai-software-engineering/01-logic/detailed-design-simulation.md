# 详细设计：模拟驾驶与飞行

## 1. 模式枚举

| 模式 | 说明 |
|------|------|
| Vehicle | 地面车辆，沿地形/道路约束（可选） |
| Aircraft | 固定翼/直升机简化模型 |
| Walk | 步行，碰撞可选 |
| Ship | 水面，高度锁定 |
| Rocket | 弹道/推力简化 |
| SatelliteOrbit | 轨道相机，可贴近空间站视角 |

## 2. 状态机

- **Idle** → **Active(mode)** → 每帧更新 **位置、姿态、速度** → 同步 **3D 相机**。
- 输入：**键盘** 与 **可选手柄**；键位表在 `03-ops/user-manual.md`。

## 3. 与数据

- 依赖 **DEM** 或 **椭球** 作为高度参考；无数据时平面运动并 Log 警告。

## 4. 分阶段

- 首版可提供 **占位**：仅切换模式标签与相机插值，**无**真实物理。
