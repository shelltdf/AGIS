# 详细设计：UI Shell（Win32 SDI）

## 1. 窗口层次

- 一个 **主框架** `HWND`：`WS_OVERLAPPEDWINDOW`，含菜单、可选 Rebar/工具栏、客户区。
- **客户区** 水平分割：**左侧** 图层 Dock 宽度可拖拽；**右侧** 为 **地图宿主**（子窗口或直接客户区绘制，实现选型见物理规格）。
- **状态栏**：`STATUSCLASSNAME`，多分区（消息、坐标、比例、缩放）。

## 2. Dock 与分割条

- 左侧为 **Dock Area**：**折叠按钮条**（可选）与 **图层树视图** 并列；全折叠时分割条行为与 `window-gui-documentation` 规则一致（不可拖或透明占位，由物理规格固定）。
- 分割条光标：`IDC_SIZEWE`（左右分割）。

## 3. 菜单结构（最小）

- **文件**：新建、打开、保存、保存为、退出。  
- **视图**：2D/3D 切换、全屏（可选）、刷新。  
- **图层**：添加数据源子菜单（占位）。  
- **语言**：中文 / English（分项）。  
- **主题**：跟随系统、浅色、深色（分项）。  
- **帮助**：帮助信息（F1）、关于。

## 4. 2D 画布 overlay（与概念一致）

- **左上**：快捷键说明面板（可折叠，默认折叠）。  
- **左下**：比例、±、适应/原点/还原。  
- **右上**：要素/图层类型可见性面板（可折叠，默认展开）。  

具体控件 ID 与消息映射见 `02-physical/gis-desktop-win32/spec.md`。

## 5. Log 窗口

- 状态栏点击或菜单打开 **模态/非模态** 对话框，多行只读编辑框 + **复制全部** 按钮。
