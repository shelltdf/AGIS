# 详细设计：2D / 3D 渲染

## 1. 模式

- **默认 2D**：正射视图，仿射变换矩阵（平移、缩放、旋转可选）。  
- **3D**：透视投影、地形、矢量挤出、点云与模型绘制；**3DGS** 用专用渲染路径（路线图）。

## 2. 输入（2D）

- **中键拖拽**：平移。  
- **滚轮**：以 **指针下画布点** 为锚缩放。  
- **左下按钮**：适应 / 原点 / 还原（语义与物理规格一致）。

## 3. 切换

- **视图菜单** 或工具栏切换 **2D | 3D**；保存当前视口范围与模式到工程。

## 4. 性能

- 大栅格：分块与视域裁剪；LOD 用于 3D；**不在首版**强制实现全部优化，仅架构预留。
