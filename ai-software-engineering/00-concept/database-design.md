# AGIS 存储与数据设计（广义数据库）

## 1. 工程与会话

- **工程文件**：保存图层列表、数据源 URI、样式、视口范围、CRS、模拟状态快照等；格式在物理层 `spec.md` 中定义为版本化结构（首版可为占位 `.agisproj` JSON）。
- **脏数据**：未保存更改在 UI 与文档模型中标记 dirty，与 `03-ops` 用户说明一致。

## 2. 缓存与临时文件

- **瓦片缓存**：TMS/在线图层本地缓存目录（路径与清理策略在运维文档中说明）。
- **金字塔与概览**：大栅格可选金字塔缓存，避免阻塞 UI（分阶段）。

## 3. 空间索引与元数据

- **矢量**：文件侧依赖 OGR；内存侧可建四叉树/R-tree（实现阶段选型）。
- **点云/三维**：包围体、LOD 元数据与文件路径关联，详见 `01-logic` 专项详细设计。

## 4. 外部格式

- **Shapefile、GeoJSON、GeoPackage** 等由 GDAL/OGR 读写时，以外部规范为准，不在本仓库维护自有 IDL 版本链，除非未来定义 **自有** 扩展字段层。
