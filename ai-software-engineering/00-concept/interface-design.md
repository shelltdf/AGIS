# AGIS 接口设计（程序间接口）

> **范围**：仅 **API / RPC / IPC / 插件边界** 等程序间接口。**不含** 窗口、菜单、快捷键等人机交互（见 `product-design.md` / `software-design.md`）。

## 1. 插件与扩展（路线图）

- **插件宿主接口**：未来可导出 C ABI 或 COM 风格接口，用于数据源驱动、渲染器、分析工具；首版无公开插件 ABI，本文档保留章节占位。

## 2. 自动化与脚本（路线图）

- **脚本宿主**：可选内嵌脚本引擎调用应用对象模型；未实现前标注「未提供」。

## 3. 进程间与网络服务（路线图）

- **本地 IPC**：可选命名管道/套接字供外部工具推送图层或视口更新。
- **HTTP**：若提供小型本地 REST 服务，端点与载荷在后续版本写入本节。

## 4. 当前版本声明

主程序 **gis-desktop-win32** 首版 **不暴露** 对外网络 API；与 GDAL/TMS 等交互均在进程内通过库调用完成，不形成独立 RPC 规格。
