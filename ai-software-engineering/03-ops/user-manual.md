# AGIS 用户说明（首版）

## 1. 环境前提

- **操作系统**：Windows 10/11 x64。  
- **首版**：无额外运行时依赖（不捆绑 GDAL）。

## 2. 启动与退出

- 运行 `agis_desktop.exe`（构建输出目录见开发者手册）。  
- **文件 → 退出** 或关闭主窗口结束程序。

## 3. 界面（SDI）

- **主窗口**：单文档界面，一个顶层窗口承载当前会话。  
- **左侧**：图层管理区域（首版为占位列表）。  
- **中央**：地图显示区，默认 **2D**；可通过 **视图** 菜单切换到 **3D**（首版为模式指示，完整 3D 管线后续版本提供）。  
- **底部**：状态栏显示就绪信息与提示。

## 4. 视图模式与地图操作

- **视图 → 2D 模式 / 3D 模式**：切换当前视图模式；2D 为 GDAL+GDI 地图（需带 GDAL 的构建）。  
- **图层 → 添加数据图层**：选择栅格或矢量文件（如 GeoTIFF、Shapefile、GeoJSON、GPKG 等），图层将出现在左侧列表并在地图中绘制。  
- **中键拖拽**：平移地图。  
- **滚轮**：以鼠标位置为锚进行缩放。

## 5. 语言与主题

- **语言 → 中文 / English**：首版可能仅提示「待实现」，以实际程序为准。  
- **主题**：跟随系统 / 浅色 / 深色（首版占位）。

## 6. 日志

- **视图 → 日志** 打开日志窗口，可复制全部文本。  
- 亦可 **双击状态栏** 打开（若当前构建已启用）。

## 7. 帮助

- **帮助 → 数据驱动说明**：打开只读说明窗口，内容包括：AGIS 图层类型（GDAL 本地文件、TMS/XYZ、WMTS、ArcGIS REST 等）、如何输入数据源（菜单路径、文件扩展名示例、URL 示例）、GDAL 格式与构建差异说明，以及**当前构建已注册的 GDAL 驱动列表**（运行时枚举）。  
- **帮助 → 关于**：在日志中写入版本与简介（开发版行为以程序为准）。  
- **F1**：打开与「数据驱动说明」相同窗口。

## 8. 后续版本

- 图层驱动（GDAL、TMS 等）、导入 DEM/矢量/点云/三维、地理配准与模拟驾驶等功能将在后续版本逐步提供，详见产品设计与发布说明。

## 9. 数据转换使用说明

- 打开路径：`工具 → 数据转换`。
- 上半区：左侧选择输入类型与子类型，中间设置转换参数，右侧选择输出类型与子类型。
- 输入与输出都支持路径文本框 + 浏览按钮（可选文件或目录，按当前类型变化）。
- 「?」说明按钮：类型与子类型旁的问号可查看详细解释。
- 命令行预览：进度条上方文本框**第一行**即为与「开始转换」相同的完整单行命令，下方为分项说明；「复制命令」仅复制该行。
- 执行后：下方显示进度、状态和本次任务日志（仅当前任务上下文）。

## 10. 数据预览与 3D 预览

- 输入/输出区均有「预览」按钮，点击后弹出预览方式菜单：
  - 内置预览（默认）
  - 系统默认打开
  - 选择其他应用
- 若数据类型为模型，选择内置预览会打开 3D 窗口：
  - 支持 OpenGL / DirectX11 切换
  - 左键拖拽旋转，滚轮缩放
  - 支持「重置视角」「适配模型」
  - 实时显示 FPS、内存/显存估算、模型与材质统计信息
  - 若存在 `map_Kd`，可点击「预览贴图」打开贴图文件
